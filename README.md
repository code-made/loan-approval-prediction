# Loan Approval Prediction

## Overview
This project predicts whether a loan application will be approved using machine learning classification algorithms.

The dataset contains applicant details such as gender, marital status, income, education, loan amount, credit history, and employment status.

## Objective
The main objective of this project is to:
- Perform exploratory data analysis (EDA)
- Handle missing values
- Apply preprocessing techniques
- Train and compare machine learning models
- Predict loan approval status

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Machine Learning Models Used
- Gradient Boosting Classifier
- Random Forest Classifier
- Decision Tree Classifier
- K-Nearest Neighbors
- Support Vector Machine (SVM)

## Features
- Data preprocessing
- Missing value handling
- Data visualization
- Model comparison
- Loan approval prediction

## Results
Gradient Boosting and Random Forest achieved the best performance among the tested classification models.

Credit history, applicant income, and loan amount were important factors influencing loan approval.

## Conclusion
This project demonstrates how machine learning can help financial institutions automate loan approval prediction and improve decision-making efficiency.
